## Development

### Setup Development Environment

```bash
# Clone repository
git clone https://github.com/neuraldevelopment/<package_name>.git
cd <package_name>

# Create virtual environment
python -m venv .venv
source .venv/bin/activate

# Install in development mode
pip install -e ".[dev,test,docs]"

# Install pre-commit hooks
pre-commit install
```

### Running Tests

```bash
# Run all tests
pytest

# Run with coverage
pytest --cov=<package_name> --cov-report=html

# Run specific test file
pytest tests/test_specific.py

# Run with verbose output
pytest -v
```

### Code Quality

```bash
# Format code
black src/ tests/

# Sort imports
isort src/ tests/

# Linting

# Type checking

# Run all quality checks
pre-commit run --all-files
```

### Building Documentation

```bash
# Build documentation

make html

# View documentation
open _build/html/index.html
```

## Troubleshooting

### Common Issues

**Import Error**
```bash

```
Solution: Ensure package is installed with `pip install -e .`

**Test Failures**
```bash

```
Review test output and fix failing tests.

**Type Check Errors**
```bash

```
Add missing type hints or fix type mismatches.
