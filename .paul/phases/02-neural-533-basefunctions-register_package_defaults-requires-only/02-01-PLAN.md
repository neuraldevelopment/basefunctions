## Goal
Simplify `register_package_defaults` to accept only `package_name` — path resolution moves inside the method via `get_runtime_config_path`.

## Output

- `test_config_handler.py` with updated tests (mock `get_runtime_config_path`)
- `basefunctions/__init__.py` caller simplified

</objective>
