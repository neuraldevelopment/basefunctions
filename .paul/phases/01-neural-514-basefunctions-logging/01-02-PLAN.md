<objective>
## Goal
Audit the `config` and `events` subpackages for logging quality: remove all debug log calls and ensure appropriate error/warning log messages exist at every significant error or exception point.

## Purpose
Extends the cli audit pattern to the two infrastructure subpackages. The events subpackage is the core message bus — unlogged errors in corelet workers and rate limiters are invisible in production. The config subpackage misuses `logger.critical` for recoverable file operation failures.

## Output
Modified source files in `src/basefunctions/config/` and `src/basefunctions/events/` with:
- All `logger.debug(...)` calls removed
- `logger.critical(...)` demoted to `logger.error(...)` where critical level is inappropriate
- All exception handlers and validation raises covered by `logger.error(...)` or `logger.warning(...)`
</objective>
